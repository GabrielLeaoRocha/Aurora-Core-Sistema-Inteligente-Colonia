# Aurora-Core-Sistema-Inteligente-da-Colonia
O sistema se chama Aurora Core e simula a central de inteligência da colônia. Ele lê dados de telemetria (via CSV), organiza as informações em estruturas hierárquicas, indexa as leituras em uma BST para busca eficiente, aplica regras de decisão lógica, realiza previsão de energia por regressão linear e gera um relatório de status da colônia.
